Start Quickly

Xing creates a Ruby On Rails API server and Angular JS application with a single command. A second command launches your development environment in an integrated console. Skip months of time lost to build systems, managing processes, and boilerplate code.

Keep It Simple

Jump headfirst into developing next-generation Single Page Applications without drowning in the deep end. Xing offers a clear path for all aspects of building an app: tools to build an API, file structure for your frontend, and a powerful client-side data library.

Plan Ahead

Xing uses the best tools of today, plus advanced features not found anywhere else. Your entire API will be hypermedia-enabled by default, the first framework where your javascript client doesn't even need to know your API servers URL's ahead of time.

Getting Started

Install and run Xing in 5 easy commands: 


gem install xing-framework

xing new book-app

cd book-app

rake initialize

rake develop
